# Image to spwn

## Dependencies
[Spwn language](https://github.com/Spu7Nix/SPWN-language) >=0.6 (0.7 recommended)
Python >=3.6
Python libraries:
- Colorama
```sh
python -m pip install colorama
```
- Pillow
```sh
python -m pip install Pillow
```

## Running

```sh
# Any platform
python3 img.py <UrImagePath>
spwn -b main.spwn
```
